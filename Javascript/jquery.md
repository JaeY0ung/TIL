# JQuery

### id="text"인 element의 html값(innercontent) 바꾸기

```html
<script>
    $('#test').html('안녕');
</script>
```
---------------------------------------------------
### id="text"인 element의 html값(innercontent) 출력하기

```html
<script>
    $('#test').html();
</script>
```
---------------------------------------------------
### id="text"인 element의 스타일(color을 red로) 변경하기

```html
<script>
    $('#test').css('color', 'red');
</script>
```
---------------------------------------------------
### id="text"인 element의 다른 속성(attribute) 변경하기

```html
<script>
    $('#test').attr('src', '/asdsaf/asdfa');
</script>
```
---------------------------------------------------
### id="text"인 element에게 class 주기

```html
<script>
    $('#test').attr('class', 'yellow-bg');
</script>
```
